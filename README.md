# predictiveIT Align

Strategic IT Alignment Platform — unified vCIO/TAM platform for standards assessments, asset lifecycle, technology roadmaps, EOS planning, CSAT tracking, and report building.

## Stack
- **Backend:** Node.js + Express
- **Frontend:** React + Vite
- **Database:** PostgreSQL 15
- **Backend hosting:** pitai-app01 (:3002)
- **Frontend hosting:** pitai-web01 (nginx → align.predictiveit.ai)
